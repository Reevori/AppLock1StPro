# AppLock1StPro

Herramienta personal para el bloqueo de aplicaciones en Windows.

## Cómo usar
1. Descargar el archivo `App Locker.exe`.
2. Ejecutarlo con permisos de administrador para asegurar el bloqueo de procesos.

## Notas
- Este repositorio es de uso personal y privado (de momento).
- Al ser un ejecutable que interactúa con procesos del sistema, es posible que Windows Defender lo detecte como sospechoso; simplemente hay que darle a "Ejecutar de todas formas".
